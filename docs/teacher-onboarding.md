# Teacher Onboarding Guide - EduBot

## Quick Start (5 Minutes)

### Step 1: Get Access
1. Make sure you have the bot: @osee_IBT_IELTS_tutor_bot
2. Send `/teacher [TEACHER_CODE]` to become a teacher
3. Contact @oseeadmin to get your teacher code

### Step 2: Create Your Class
1. Open the bot
2. Send `/admin`
3. Click "Create New Class"
4. Give it a name (e.g., "TOEFL Prep Batch 1")
5. Get the invite code

### Step 3: Invite Students
1. Share the invite code with students
2. Students send `/join [CODE]` to the bot
3. Students appear in your dashboard

### Step 4: Set Up Group (Optional)
1. Create a Telegram group for your class
2. Add the bot to the group
3. Send `/addclass` in the group
4. Bot will post daily quizzes automatically

## Troubleshooting

### Student Can't Join
- Check invite code is correct
- Student must send `/join CODE` directly to the bot (not in group)

### Daily Quiz Not Posting
- Check bot is admin in group
- Check `/addclass` was sent in group
- Try `/addclass` again

### Student Premium Issues
- Students get 1-day free trial
- After that, they need to buy premium
- Direct them to `/premium` or WhatsApp
