1. Getting Started 🚀

1.1 Activate Your Teacher Account

Before you can manage classes, you need to activate your teacher role:

  1. Open @osee_IBT_IELTS_tutor_bot in Telegram
  2. Send /teacher [invite_code] — use the invite code provided to you
  3. You'll receive a TEACHER badge and access to teacher commands

Note: You get a 7-day free trial as a teacher. After the trial, you need to activate premium to continue managing classes.

1.2 Setting Up Your Profile

Configure your teacher settings:

Send /settings

This lets you:

  • Set your display name for students
  • Choose your class timezone
  • Configure notification preferences

1.3 Understanding Teacher Commands

As a teacher, you have access to these commands:

CommandDescription
/adminOpen the admin dashboard (web app)
/addclassCreate a new class
/broadcast MSGSend message to all students
/statsView system statistics
/pendingpaymentsSee pending student payments

2. Class Management 📚

Organize your students into classes for easier management and automated daily quizzes.

2.1 Creating a Class

To create a new class:

Send /addclass

Follow the prompts:

  1. Enter class name (e.g., "TOEFL Intensive Batch 1")
  2. Enter class description (optional)
  3. You'll receive a unique invite code
  4. Share this code with your students

2.2 Inviting Students

Students can join your class using the invite code:

Send /join CODE

Where CODE is the invite code you gave them.

💡 Pro Tip

Create different classes for different programs (e.g., TOEFL, IELTS) or batches. This way, you can send targeted content to each group.

2.3 Managing Multiple Classes

You can manage multiple classes:

  • View all classes: Use /admin dashboard
  • Switch between classes: Select class from the dashboard tabs
  • Archive old classes: Contact admin for class archival

2.4 Daily Quiz System

Each class gets an automatic daily quiz at 8 AM WIB. The quiz:

  • Contains 5 questions from your class's target exam
  • Mixes question types (Reading, Listening, Grammar, etc.)
  • Updates the class leaderboard
  • Reminds students who haven't answered

3. Monitoring Students 👀

The /admin dashboard gives you full visibility into your students' progress.

3.1 Student List View

Access the dashboard:

Send /admin

Then click the Students tab. You'll see:

  • Student name and Telegram username
  • Current level and XP
  • Questions answered today
  • Last active time
  • Premium status

3.2 Individual Student Details

Click on any student to see:

  • Progress Overview — Band score estimation, section breakdown
  • Accuracy by Section — Reading, Listening, Speaking, Writing
  • Study Streak — Consecutive days of practice
  • Weak Areas — Topics that need more work
  • Recent Activity — Last 10 questions with results

3.3 Class Leaderboard

Keep students motivated with rankings:

Leaderboard shows:

🥇 1st Place — Student with most XP this week

🥈 2nd Place — Second highest XP

🥉 3rd Place — Third highest XP

📊 Weekly updates every Monday at 8 AM WIB

3.4 Identifying Struggling Students

Watch for these warning signs:

  • ⚠️ No activity for 3+ days
  • ⚠️ Accuracy below 40%
  • ⚠️ Questions answered decreasing

Use /broadcast to send encouragement or study tips to inactive students.

8. Frequently Asked Questions ❓

Q: How many students can I have per class?

A: Standard plan allows up to 50 students per class. Contact us for higher limits.

Q: Can I use EduBot alongside my existing teaching methods?

A: Absolutely! EduBot is a supplement, not a replacement. Use it for homework, self-practice, and daily quizzes while you focus on live sessions.

Q: How do students pay?

A: Students can pay via Telegram Stars or GoPay transfer. You can also collect payments manually and activate their accounts via the admin dashboard.

Q: Can I create custom content for my class?

A: Yes! Contact admin to submit custom questions or drills. Premium teachers get priority review.

Q: What if a student wants a refund?

A: Handle refunds between you and the student directly. EduBot's payment system is separate from your student management.

Q: How do I track my revenue?

A: Use the /stats command to see your student count and billing information.

Q: Can I offer a free trial to prospects?

A: Yes! Direct them to @osee_IBT_IELTS_tutor_bot for the standard 1-day free trial.
